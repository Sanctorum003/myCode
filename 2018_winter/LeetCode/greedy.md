# 561. 数组拆分 I
> 由于我们要最大化每对中的较小值之和，那么肯定是每对中两个数字大小越接近越好，因为如果差距过大，而我们只取较小的数字，那么大数字就浪费掉了。
> 证明：首先找到最小值 x ，它跟任何其他值配对，min([x, anything]) 都会得到 x。而 anything 取次小值时对剩下的数损耗最小，所以跟 x 配对的是次小值。余下的数同理，所以最佳方案是先将数组排序，再两两配对。
