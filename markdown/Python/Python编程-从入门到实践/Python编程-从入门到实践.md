# 2-列表简介
> 用（[]）来表示列表，并用逗号来分隔其中的元素

1.索引为-1指定最后一个元素

2.在列表末尾添加元素
> .append()

3.在列表中插入元素
> .insert(位置,value)

4.删除列表中的元素
> del 列表名[位置]

> .pop() 弹出列表中的末尾元素并返回

> .pop(位置) 弹出列表中某个位置的元素并返回

> .remove(元素名) 删除列表中某个特定元素

5.永久性排序
> .sort() 

> .sort(reverse = True) 逆字典顺序

6.临时排序
> .sorted() 

> .sorted(reverse = True) 逆字典顺序

7.倒着打印列表
> .reverse() 永久性改变

8.确定列表长度
> len(列表)
